agent-reach opencode cursor
多平台互联网调研与内容获取工具。支持 13 个平台（小红书、Twitter/X、B站、Reddit、V2EX、LinkedIn、YouTube、GitHub 代码搜索、小宇宙播客、雪球股票行情、RSS 订阅、任意网页 URL）。多后端路由（OpenCLI / 各平台专用 CLI / API），其中 6 个渠道零配置。
互联网调研/搜索/查找场景。按平台分类路由（搜索/社交/求职/开发/网页/视频）。运行 agent-reach doctor --json 查看各平台当前使用的后端。
 agent-reach doctor
docx opencode
端到端 Word 文档创建与操作。将 Markdown 或结构化文本转为精美的 Word 输出。填充可复用模板（{{token}} 或参考文档）。应用 CJK 排版默认值。使用 docx-js 从零生成定制文档。底层 OOXML 修补，支持修订标记和批注。
触发场景：Word 文档、.docx 文件、报告、备忘录、合同、会议纪要、Markdown 转 Word、中文排版、修订标记、批注、OOXML 编辑。
 docx
xlsx opencode windsurf
读取、编辑、创建和修复电子表格。支持 .xlsx、.xlsm、.csv、.tsv 格式。添加列、计算公式、应用格式、创建图表、清洗杂乱数据。在表格文件为主要输入或输出时自动触发。
触发场景：.xlsx/.csv/.tsv 文件、创建电子表格、编辑 xlsx、csv 转 xlsx。不适用于：Word 文档、HTML 报告、数据库管道。

触发场景：任何涉及 .pdf 文件的操作——作为输入或输出。合并 PDF、提取文字、OCR 扫描件、填写 PDF 表单、加密 PDF。
